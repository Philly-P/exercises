Lists

There are three kinds of lists: ordered, unordered, and definition. List are wrapped in a list container.

Oreder list

  1. Item one
  2. Second item
  3. Another item
  4. Last one

Unorder lists

  • First same
  • Not again
    • Lists can be
    • Nested to
    • Arbitrary depth
  • Henceforth

Definition list

Definition term
Definition details include more information.
Lists, by definition, include more than one item

Links

Links use the a tag and display inline. They must have a href attribute and can use target to open in a newtab.

Links have different psuedo-class states, if they're hovered or have been visited, for example.

Images

Images use the img tage, which is self closing, and they might be inside a figure tage. Images use the src=attribute to point to the source,which could be anywhere. Images display inline by default
